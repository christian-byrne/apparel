Apparel | Help

Deposit clothing items. Use the Add Item page in the navbar. Log an item into your wardrobe by entering some details about it. Most importantly, the colors.

Record clothing combinations. Record/log outfits using the Generate Outfit page in the navbar.

Start by searching items using either the general search or the filtered search below that. The filtered search requires you to select a field first and then only applies search to the specified field. If search results are already displayed, it applies the filter to them.
Use the clear search results button to the right of the filter

Add items from the search results, and they will go into the item queue at the top right of the page. Items in the queue can be expanded or removed by clicking on them.

Finally, add details about the outfit at the bottom of the page. Here you can also see a visual representation of the outfit on the mannequin given the colors of the items in the queue. Then, submit.

Use the app's style suggestions to generate outfits. The generate outfit page can also be used to create oufits on the fly (in addition to just recording them).

After a few items have been added to the queue, click the Suggest Next Item button below the search bar and a color palette suggestion tool will be opened in another tab.

On that page, lock the selected colors using the lock icon, then add new colors and cycle through suggestions using the space bar.

Then, use those palette suggestions to inform your selection of further items in the outfit.

View all the clothing items in your wardrobe. Click the Wardrobe tab in the navbar to view your items.

The items from your wardrobe will be organized into the tabs based on the categories of the items. If a category is not specified for an item, it will be found in the Other tab.

Dismiss an item using the dismiss button, or edit the item with the edit button

Use the sidebar to complete various actions on the items.

Choose an outfit to decide what to wear. Click the Outfits portal to see your collection.

Outfits contain references to their items, which can be inspected/expanded by clicking on them.

For easy data entry or backing up saved work. Make sure data is in an .xlsx file. The colors are parsed based on the background color of the color fields of the records.

Use the search bar in the masthead to search for features/pages.
